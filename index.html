How to make payments fully active (quick checklist)

  1. PayPal.me बनाएँ: https://www.paypal.me/YourName — इसे paypalInput में रखें।
  2. UPI QR: किसी UPI प्रोवाइडर से QR बनवाएँ और image URL को qrInput में रखें (या GitHub assets में अपलोड करें)।
  3. Email: कामकाजी ईमेल दें — यह emailInput में डालें।
  4. GitHub Releases: /assets में फाइलें डालकर Release बनाएं; फिर GitHub Releases guide देखें।

नीचे दिए फॉर्म में URL/ईमेल डालते ही साइट पर बटन सक्रिय हो जाएंगे — यह क्लाइंट-साइड सक्रियता है ताकि आप सीधे देख सकें कि क्या काम कर रहा है।
